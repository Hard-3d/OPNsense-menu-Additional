OPNsense Additional Menu v0.1.0

Установка:
cd /
unzip -o /root/opnsense-additional-menu-v0.1.0-root.zip
chmod 755 /install.sh
/install.sh

После установки:
Logout -> Login

Дополнительно
- Ethname
- GeoIP update
- Check VPN status
- Check WAN
- Update

Для GitHub:
- распакуйте архив в локальную папку репозитория;
- сделайте git add/commit/push;
- создайте Release v0.1.0;
- загрузите этот же root ZIP как release asset.
